# LLM Chat - iOS

Chat with local LLM models using Ollama. Connect to your local Ollama server and chat with powerful AI models.

## Features
- Local Ollama model support
- Chat with any Ollama model
- Conversation history
- Model picker
- Settings and customization

## Tech Stack
- SwiftUI
- Xcode

## Build
Open `LLMChatApp.swift` in Xcode and build.
