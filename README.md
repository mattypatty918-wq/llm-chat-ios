# NexusAI Codex - GitHub Codespace

## Quick Start
1. Open this repo in Codespaces
2. The environment auto-installs (gotty web terminal + API)
3. Access at: https://8081-{PORT}.{{your-codespace}}.github.dev

## Manual Start
```bash
bash /opt/nexusai/start.sh
```

## API Endpoints
- POST /chat - AI chat
- GET /health - Health check
